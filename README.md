# Udacity Front-End Web Developer Nanodegree
# Project 2 - Memory Game Project
## Summary:
 This is a nanodegree program second project.

## The application offers the following features:
- Timer is ticking
- Restart game
- If card matches, cards color turns to green
- If not, cards  close up again.
- Congrats modal when game is over


## Technologies Used In Application:
- HTML
- Javascript for functionality
- Google Fonts - (https://fonts.googleapis.com/css2?family=Indie+Flower&display=swap)
- Bootstrap 4.5 - (http://getbootstrap.com/) framework for responsive design
